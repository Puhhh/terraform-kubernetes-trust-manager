# How to use

* Change S3 values 

```bash
terraform init
```
```bash
terraform plan
```
```bash
terraform apply
```

## Optional 

* If you need to change the default values of variables, add them to the *terraform.tfvars* file.
* If the value of *custom_values* is *true*, *values.yaml* will be used. You can modify *values.yaml*.
